# Build & Run application
## Build jar
1. Склонировать репозиторий
2. Запустить скрипт сборки — `build.sh`
```bash
./gradlew build
```


## Run
Запустить скрипт `run.sh`
```bash
./gradlew build
```

## Test
Запустить скрипт `test.sh`
```bash
./gradlew test
```
